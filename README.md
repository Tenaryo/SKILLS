# Opencode_Skills

Tyler Tan's personal collection of [opencode](https://opencode.ai) skills.

## Why not oh-my-opencode or superpowers?

They're great projects, but I found them a bit too heavy for my taste. I prefer keeping a minimal set of skills that I actually use day-to-day, without all the extra bloat.

## Included Skills

- **explain-code** - Step-by-step code explanation
- **feat-workflow** - Test-driven feature development workflow
- **fix-workflow** - Bug fixing workflow with reproduction and root cause analysis
- **review-workflow** - C++ code review and refactoring
- **skill-creator** - Guide for creating new skills
